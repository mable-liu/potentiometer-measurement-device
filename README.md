
# Potentiometer Measurement Device

Custom measurement device built to estimate object dimensions using a rotary potentiometer, Arduino Uno, and a 3D-printed linkage mechanism.

## Overview
This project used angular displacement measured by a potentiometer to estimate linear dimensions through calibration and geometric relationships. The system was designed as a first-principles alternative to a commercial measurement tool.

## Tools Used
- Arduino Uno
- C++
- SolidWorks
- 3D Printing
- Rotary Potentiometer
- Sensor Calibration
- Uncertainty Analysis

## Features
- Converts potentiometer voltage into angular displacement
- Uses calibration data to estimate dimensions in millimeters
- Built with a custom 3D-printed mechanical frame and linkage
- Evaluated system uncertainty through experimental testing

## Results
- Achieved a maximum measurement error of ±1.47 mm across the tested range
- Developed calibration workflow using known reference lengths
